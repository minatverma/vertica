Vertica misc
------------

UDF SQL functions:
~~~~~~~~~~~~~~~~~~

  * is_int         - validates if argument is from type integer
  * month_name     - returns month name
  * weekday_name   - returns weekday name
  * random_ts      - returns random timestamp
  * mid_ts         - returns middle date between two dates
  * is_leap_year   - reurns if year is leap
  * season         - reurns season = {winter,spring,summer,autumn}
